About

Eon is a simple game designed to test Clojure beginners in their ability to evaluate basic Clojure forms. I wrote it to learn what the ClojureScript fuss is all about (it's awesome and fun).

Eon consists of several rounds of 10 questions per round. Each question is initially worth 100 points. Entering an incorrect question drops the point value by 10. There are no time limits.

Eon uses (and likely abuses!) the following projects:

  • Reagent
  • clojurescript.test

The source can be found on GitHub.
